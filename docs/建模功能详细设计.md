模块1：问题探索舱（步骤1）
上传问题
理解与讨论问题过程，原理讲解与动画表达
最终根据讨论结果，生成解决方案（第一步结论数据）


模块2：模型搭建室（步骤2）
学习模型为第一目标

两种模式：研究性数学模型（仅有目标） / 数据挖掘（明确数据与目标）

a.模型库：多种模型列表展示
    模型原理讨论
    模型对于问题的解决方法说明和结果

b.数据分析
    模型数据需求讨论（数学模型类数据可能需要创造，也可能可以外部获得）
    数据研究：
        数据上传
        数据讨论
    (未来后端可以配备数据集，或者数据获取服务)


模块3：计算验证站（步骤3）
    讨论模型的计算过程
    实现：引导学生利用编程实现模型代码
    
    讨论验证模型的方案，数据可视化的方案
    结果：验证模型的代码


模块4：成果展示厅（总结与分享）
    提交结论数据
    html版本的数据展示

    论文生成 Agent


slogon：在 AI 时代，你的价值不再取决于你记住了多少公式，而取决于你能提出多深刻的问题，以及你整合资源解决问题的逻辑。
